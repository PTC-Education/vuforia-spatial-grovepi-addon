This is the hardware interface to connect to your Arduino board.

To use your Arduino with the Spatial Toolbox, you need to navigate within your Arduino IDE to the following menu:
"Sketch > Inlude Library > Add .zip Library..."

With the resulting dialog, open the following Zip File:
vuforia-spatial-basic-interfaces-addon/interfaces/arduinoUno/Vuforia-Spatial-Arduino-Library.zip
